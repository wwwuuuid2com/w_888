<p>微社区动态    即时聊天    私聊    群聊    好友系统<p>
<p>礼物系统    直播系统    缘分匹配    金币系统<p>
<p>1.删除config/install.lock 输入程序所在网址即可自动安装    2.php需要开启https支持    3.安装完成 删除install.lock<p>
